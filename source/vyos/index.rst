.. _vyos:

VyOS
====

VyOS is an open-source network operating system that provides a powerful and flexible platform for routing and managing network traffic. It is based on the Debian Linux distribution and includes a wide range of networking tools and protocols such as BGP, OSPF, VRRP, QoS, NAT, and VPN. VyOS is designed to be installed on hardware or virtual machines and can be easily customized to fit the specific needs of an organization.
