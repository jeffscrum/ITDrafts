VMAX. Тестовый CallHome
=======================

Для проверки работы функции CallHome

1. В SymmWin выбираем: Configuration -> Site information -> СallHome.
2. Жмем кнопку "Test SymmWin Call Home now".
3. Нажимаем кнопку "ОК" в появившемся окне.
4. Смотрим результат.
