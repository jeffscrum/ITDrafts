Spares Kit Components for Exadata
=================================

* FR -- Full Rack
* HR -- Half Rack
* QR -- Quarter Rack
* ER -- Eighth Rack


Oracle Exadata Database Machine X11M
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Note: No spares kit is included with Oracle Exadata Database Machine X11M.  
Spare hard disk drives and flash devices may be purchased separately.

Oracle Exadata Database Machine X10M
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Note: No spares kit is included with Oracle Exadata Database Machine X10M.  
Spare hard disk drives and flash devices may be purchased separately.

Oracle Exadata Database Machine X9M-2 and X9M-8
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  1 x 18 TB disk for High Capacity storage servers only
-  1 x 6.4 TB NVMe PCI Flash card

Oracle Exadata Database Machine X8M-2, X8M-8, X8-2, and X8-8
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  1 x 14 TB SAS disk for High Capacity storage servers only
-  1 x Flash Accelerator F640 or F640v2 PCIe card

Oracle Exadata Database Machine X7-2 and X7-8
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  1 x 10 TB 7.2k RPM SAS disk for High Capacity storage servers only
-  1 x Flash Accelerator F640 PCIe card

Oracle Exadata Database Machine X6-2
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  1 x 8 TB 7.2k RPM High Capacity SAS disk
-  1 x flash accelerator F320 PCIe card for high capacity storage servers only

Oracle Exadata Database Machine X5-2
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  1 x 8 TB 7.2k RPM High Capacity SAS disk
-  1 x flash accelerator F160 PCIe card for high capacity storage servers only
-  1 x 1.6 TB solid-state drive for extreme flash storage servers only

Oracle Exadata Database Machine X5-8 and X6-8
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

-  1 x 8 TB 7.2k RPM High Capacity SAS disk
-  1 x flash accelerator F160 PCIe card for high capacity storage servers only
