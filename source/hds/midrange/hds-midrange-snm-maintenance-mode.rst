HDS SNM Maintenance Mode
========================

#. Заходим в SNM через GUI. Ставим курсор на "Add new array".
#. Нажимаем комбинацию клавиш :kbd:`Ctrl` + :kbd:`Shift` + :kbd:`e`.
#. После чего "Operation Mode" сменится на "Maintenance Mode".
