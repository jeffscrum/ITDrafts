HDS Midrange Maintenance IP
===========================

Default IPs for maintenance port are:

-  10.0.0.16 (CTL#0)
-  10.0.0.17 (CTL#1)

If this network is used for management array, IPs will be next:

-  192.168.0.16 (CTL#0)
-  192.168.0.17 (CTL#1)
